Wisconsin Farm Parcels

This is a map I made for a friend as a gift, detailing all the farm land their family owns in Wisconsin. In the process of making this map, I learned how to use a new application called Eduard, which helped create the shaded relief I used in the basemap. This was a super fun map to make and I learned a new way to create shaded relief for any location around the world. What a great experience!
